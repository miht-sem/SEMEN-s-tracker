## Working with SEMEN's tracker

* Press `(⇧ + ⌘ + P)` \ `(Ctrl + Shift + P)` to open the Command Palette.
* Write one of available commands:
    * `Set Current Task`
    * `Clear Current Task`
    * `Set Timer`
	* `Toggle Timer`
    * `Clear Timer`
* Press `(↵ Enter)`
